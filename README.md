# ASW Form Builder
The **ASW Form Builder** is allow you to create forms, resume and about us page to build as a JSON Schema which renders as a **Form, pages, and templates** in your application.
This form builder is designed with `Angular` latest version.

## [API Demo](https://asoftwareworld.github.io/ASW-Form-Builder/#/)

## List of Controls available
- Header
- Autocomplete
- Textbox
- Text area
- Datepicker
- Select
- Multi select
- Radio button
- Checkbox button
- Button
- Paragraph
- Divider
- Slide toggle
- Grid Component **Coming soon**
- Material icon **Coming soon**
